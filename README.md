## 📖 Sobre o projeto

O **Simulador de Phishing - SuperOfertas Brasil** foi desenvolvido com o objetivo de conscientizar usuários sobre golpes de engenharia social e ataques de phishing.

A aplicação simula uma página promocional semelhante às encontradas na internet, permitindo demonstrar como criminosos utilizam interfaces convincentes para coletar informações pessoais.

> **Este projeto possui finalidade exclusivamente educacional e de conscientização em Segurança da Informação. Nenhuma informação é utilizada para fins maliciosos.**

---

## 🎯 Objetivos

- Demonstrar como golpes de phishing funcionam.
- Incentivar boas práticas de segurança digital.
- Conscientizar usuários sobre engenharia social.
- Servir como material de treinamento em cibersegurança.

## 🖥️ Executando o projeto

Clone o repositório

```bash
git clone https://github.com/Joda33/realish-mercado-play-main--1-.git
```

Entre na pasta

```bash
cd realish-mercado-play-main--1-
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```

A aplicação estará disponível em

```
http://localhost:5173
```
